# Adversarial-Machine-Learning-applied-to-IDS
Deep Learning-based NIDS trained on UNSW-NB15, audited via Adversarial Machine Learning. Implements White-Box (CPGD) and Black-Box (PSO) attacks with physical network constraints — no payload shrinking, no time travel. Hardened via Adversarial Training, dropping CPGD ASR from 53% → 2.4% and PSO ASR from 98% → 49.6%. Built with PyTorch.
